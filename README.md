# Login Page 🔐

A simple and responsive **Login Page** built with HTML and CSS. Users can enter their username and password to access a website. Features a clean design, user-friendly interface, and works seamlessly on both desktop and mobile devices.

---

## Features

- **Clean & Modern Design:** Minimalist, easy-to-use interface.  
- **Responsive Layout:** Works perfectly on desktops, tablets, and mobile devices.  
- **Input Fields:** Username and password inputs styled with CSS.  
- **User-Friendly Experience:** Simple login interface for quick access.
